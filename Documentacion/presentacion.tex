\begin{frame}{El Problema}
\begin{itemize}
    \item \textbf{Cansancio:} Los conserjes se cansan de mirar pantallas todo el día y se les pasan cosas.
    \item \textbf{Cámaras pasivas:} Las cámaras de seguridad típicas solo sirven para ver qué pasó después del robo, no para evitarlo.
    \item \textbf{Desorden:} Es muy difícil controlar a las mascotas sueltas o los autos mal estacionados.
    \item \textbf{Falta de información:} Los vecinos no se enteran de las cosas en tiempo real.
\end{itemize}
\end{frame}

\begin{frame}{Nuestra Solución}
Creamos el \textbf{Condominio Smart System}, que es súper rápido y funciona solo:
\vspace{0.5cm}
\begin{itemize}
    \item \textbf{Placas ESP32:} Pusimos microcontroladores que cuestan solo 30 soles, lo cual hace que sea súper barato de implementar.
    \item \textbf{Visión con IA:} El sistema mira el video de las cámaras y se da cuenta si es una persona, un auto o un perro.
    \item \textbf{Respuestas rápidas:} Si pasa algo malo, suena la alarma solita y le avisa al guardia por la web al instante.
\end{itemize}
\end{frame}

\begin{frame}{¿Quiénes usan el sistema?}
Hicimos que cada persona tenga su propia vista:
\vspace{0.5cm}
\begin{itemize}
    \item \textbf{Administrador:} Puede ver todo, cambiar reglas y revisar multas.
    \item \textbf{Recepción:} Ve la lista de problemas para ayudar a los vecinos o cobrar multas.
    \item \textbf{Seguridad:} Solo le llegan las alertas urgentes para que vaya rápido al lugar.
    \item \textbf{Residente:} Entra a ver si le llegó alguna notificación o multa a su casa.
\end{itemize}
\end{frame}

\begin{frame}{Arquitectura y Tecnologías}
Todo está hecho con herramientas modernas:
\begin{itemize}
    \item \textbf{Frontend:} React y TailwindCSS (queda muy bonito).
    \item \textbf{Backend:} Spring Boot en Java con seguridad JWT.
    \item \textbf{La IA:} Un script en Python que usa OpenCV para detectar cosas.
    \item \textbf{Hardware:} Las plaquitas ESP32 (de 30 soles) que mandan datos por MQTT.
    \item \textbf{Base de Datos:} PostgreSQL, todo corriendo en Docker.
\end{itemize}
\end{frame}

\begin{frame}{¿Por qué esto sirve?}
\begin{itemize}
    \item \textbf{Más seguros:} La IA no parpadea, así que pilla a los ladrones antes de que entren.
    \item \textbf{Guardias más útiles:} Los conserjes pueden hacer sus rondas o ayudar en portería en vez de ver teles aburridas.
    \item \textbf{Multas justas:} Si alguien mete a su perro a la piscina, el sistema lo pilla automáticamente y sin favoritismos.
\end{itemize}
\end{frame}
